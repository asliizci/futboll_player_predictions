# PREDICTING SOCCER PLAYER VALUES
This project presents a machine learning model and an intuitive interface to predict football players' market values using their skills and attributes. It provides clubs with a
data-driven tool to make informed transfer decisions, evaluate players effectively, and optimize their budgets.
